AP5 - Réalisation Professionnelle

Déploiement GLPI & Plateforme Moodle

2025/2026 - Atelier Professionnel (Labo)

📋 Description de la Réalisation Professionnelle

Ce projet consistait à déployer une infrastructure complète de services informatiques pour un organisme de formation. Deux solutions principales ont été mises en place :

  • GLPI : Solution de gestion de parc informatique et de gestion de tickets d'incidents
  • Moodle : Plateforme de e-learning avec authentification et accès sécurisé à distance

L'infrastructure repose sur un hyperviseur ESXi VMware Horizon, avec des VM dédiées, une base de données MariaDB, une supervision via Zabbix et des sauvegardes vers un NAS.

🖥️ Solutions Déployées

📊 GLPI - Gestion de Parc et Tickets

  • Gestion centralisée du parc informatique
  • Gestion des tickets d'incidents et demandes
  • Plugin FusionInventory pour inventaire automatique
  • Base de données MariaDB dédiée
  • Lien API avec Zabbix pour la supervision

📚 Moodle - Plateforme E-learning

  • Plateforme pédagogique accessible à distance
  • Séparation des services Web/Base de données
  • Authentification sécurisée

  • Séparation des réseaux (DMZ)

🏗️ Architecture Technique

Hyperviseur : ESXi VMware Horizon

  • VM1 : Serveur Web (Moodle) + GLPI
  • VM2 : Serveur Base de données MariaDB
  • VM3 : Serveur Supervision Zabbix
  • NAS : Sauvegarde des données

Phases de déploiement

  • 1er temps : Mise en place de la solution Web/SGBD sur un unique serveur
  • 2ème temps : Séparation des services (Web/BDD)
  • 3ème temps : Séparation des réseaux (VLAN, DMZ publique/privée)
  • 4ème temps : Implémentation du protocole sécurisé HTTPS
